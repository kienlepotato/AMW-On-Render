This is a node.js project that implements a basic html,css,ejs and js script frontend
and a node.js backend.
this project uses a SQLite databse file 
To use this project you need to have node.js installed
To run this proeject simply go to terminal, change directory to that containing the server.js file and enter the command: node server.js

RUN THIS TO CREATE EXE:
